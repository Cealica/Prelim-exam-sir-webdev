A LEXICON FOR BEGINNERS









HTML

HTML

An Acronym for Hyper text Markup Language, this computer "language" allows you to create web content, navigate around the web, and define certain types of content.









CSS

CSS

An acronym for Cascading Style Sheet, this digital technology allows you to control the appearance placement, and function certain elements on your web pages, like page position, shape, size, and colors.

SNIPPETS

Short reusable sections of programming that allows you to minimize repitition and clarity function.
